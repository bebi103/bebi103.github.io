bebi103.image.imshow(im, cmap=None, length_units='pixels', interpixel_distance=1.0, colorbar=False, no_ticks=False, flip=True, return_im=False, saturate_channels=True, min_intensity=None, max_intensity=None, display_clicks=False, record_clicks=False, **kwargs)ÔÉÅ

Display an image in a Bokeh figure.

Parameters

  • im (Numpy array) ‚Äì If 2D, intensity image to be displayed. If 3D, first two dimensions are pixel values. Last dimension can be of length 1, 2, or 3, which specify colors.

  • cmap (str or list of hex colors, default None) ‚Äì If im is an intensity image, cmap is a mapping of intensity to color. If None, default is 256-level Viridis. If im is a color image, then cmap can either be ‚Äòrgb‚Äô or ‚Äòcmy‚Äô (default), for RGB or CMY merge of channels.

  • length_units (str, default 'pixels') ‚Äì The units of length in the image.

  • interpixel_distance (float, default 1.0) ‚Äì Interpixel distance in units of length_units.

  • colorbar (bool, default False) ‚Äì If True, include a colorbar.

  • no_ticks (bool, default False) ‚Äì If True, no ticks are displayed. See note below.

  • flip (bool, default True) ‚Äì If True, flip image so it displays right-side up. This is necessary because traditionally images have their 0,0 pixel index in the top left corner, and not the bottom left corner.

  • return_im (bool, default False) ‚Äì If True, return the GlyphRenderer instance of the image being displayed.

  • saturate_channels (bool, default True) ‚Äì If True, each of the channels have their displayed pixel values extended to range from 0 to 255 to show the full dynamic range.

  • min_intensity (int or float, default None) ‚Äì Minimum possible intensity of a pixel in the image. If None, the image is scaled based on the dynamic range in the image.

  • max_intensity (int or float, default None) ‚Äì Maximum possible intensity of a pixel in the image. If None, the image is scaled based on the dynamic range in the image.

  • display_clicks (bool, default False) ‚Äì If True, display clicks to the right of the plot using JavaScript. The clicks are not recorded nor stored, just printed. If you want to store the clicks, use the record_clicks() or draw_rois() functions.

  • record_clicks (bool, default False) ‚Äì Deprecated. Use display_clicks.

  • kwargs (dict) ‚Äì All other kwargs passed to bokeh.plotting.figure().

Returns

  • p (bokeh.plotting.figure instance) ‚Äì Bokeh plot with image displayed.

  • im (bokeh.models.renderers.GlyphRenderer instance (optional)) ‚Äì The GlyphRenderer instance of the image being displayed. This is only returned if return_im is True.
